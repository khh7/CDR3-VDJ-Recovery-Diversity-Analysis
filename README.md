# Diversity Analysis

This repository contains downstream statistical analyses for immune receptor diversity data generated by the CDR3 V(D)J Recovery workflow.

The analysis integrates receptor diversity metrics with clinical metadata and performs:

- Clinical association testing using non-parametric statistics
- Pairwise subgroup comparisons
- Survival analysis (Overall Survival and Progression-Free Interval)
- Immune repertoire recovery landscape visualization
- Receptor composition visualization across patients

CDR3 V(D)J Recovery is a workflow for recovering T-cell and B-cell receptor CDR3 sequences from bulk RNA-seq data using alignment-based analysis methods originally described by Ferrall-Fairbanks et al. (2022).
^^ reference dr. ff github's docker container (link) - can run locally

The workflow performs end-to-end processing from:

1. RNA-seq data retrieval
2. Alignment and receptor sequence recovery
3. Per-patient repertoire quantification
4. Diversity metric calculation

This repository utilizes the diversity and repertoire summary files generated by that workflow and combines them with clinical annotations for downstream analysis.

# Customization

Several parameters are intended to be modified by the user.

## Update Input Dates

Replace hard-coded file dates throughout the script:

`2026-05-17`

with the date corresponding to your CDR3 V(D)J Recovery output files.

## Clinical Variables

Modify:

```
clinical_vars <- c(
  "TNM",
  "Stage",
  "subtype"
)
```

to analyze different clinical characteristics.
